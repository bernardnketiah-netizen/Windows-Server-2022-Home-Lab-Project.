# Windows-Server-2022-Home-Lab-Project.
Hands-on project: Installed and configured Windows Server 2022 on a VirtualBox virtual machine.
## Skills Demonstrated
- Installed and configured Windows Server 2022  
- Set up a virtual machine using VirtualBox  
- Configured Administrator account and basic network settings  
- Hands-on experience creating a lab environment for IT support  

## Setup / Steps
1. Created a VirtualBox VM with 4 GB RAM and 60 GB virtual hard disk  
2. Attached the Windows Server 2022 ISO file  
3. Installed Windows Server 2022 64-bit on the VM  
4. Configured the Administrator account and basic network settings
